# Notes for the next weekly plan

Free-form notes for the upcoming Saturday's plan. Edit this file anytime
during the week (the GitHub web UI works fine on a phone). The Saturday
workflow reads this and adjusts the plan accordingly, then prunes any
entries whose dates fall within or before the week it just planned —
so this file naturally stays clean.

## How to write a note

Anything Claude can interpret. Date-anchored notes are clearest. Examples:

- 5/1–5/2: one parent + a kid away camping, plan dinners for 3 not 5
- Friday 5/9: school potluck, dinner doesn't need to be a full meal
- All week: one parent on a work trip Mon–Wed, easy reheat-friendly meals those nights
- Trader Joe's run Tuesday — feel free to lean on TJ ingredients
- Try to use the half-bag of arborio rice in the pantry
